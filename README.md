# rythamo.qzz.io - The Rythamo Gazette

Welcome to **The Rythamo Gazette**, a premium, vintage-newspaper styled portfolio website built using Next.js 15+, TypeScript, and Vanilla CSS tokens. 

This portfolio showcases the engineering work, personal archive, and academic profile of **Vishnu Vardhan M** (widely known in the community as **Rythamo**), a CSE student at Malla Reddy Deemed to be University and a Generative AI Engineer from Gooty, AP, India.

---

## 📰 Aesthetic & Architectural Highlights

### 1. Typography & Printing Press Heritage
- **Title Banner**: Grand serif headlines utilizing `Playfair Display` mapped directly to standard headings.
- **Copy**: Highly legible Lora text and vintage monospace headers to emulate classic Linotype typesetting grids.
- **Drop Caps**: Elegant floated first-letters (`.drop-cap`) that anchor main news sections.
- **Separators**: Dual-border hair-lines and decorative columns reminiscent of 1920s print media.

### 2. Tactile Paper Textures & Framing
- **Newsprint Background**: Repeating SVG fractal noise overlay that produces a high-fidelity organic page grain on any high-resolution viewport.
- **Photographic Engravings**: Greyscale-to-color interactive framed halftone meshes mapping the provided photography of Rythamo. Hovering reveals seasonal color spectrums and detailed descriptions.

### 3. Emblematical Seal
- A minimal `"Ry_"` emblem logo designed as a circular editorial stamp, featuring outer concentric dashes and curved typography tracking our operational date (`EST. 2026`).

### 4. Direct Wire Telegram
- A live letter-to-the-editor form modeled after vintage Indian Telegraph dispatch layouts. Senders can select urgency metrics and compile text in the traditional "STOP" message format.

## 🛠️ Tech Stack & Structure

- **Framework**: [Next.js 15+](https://nextjs.org/) App Router
- **Runtime Compiler**: TypeScript & Next Turbopack
- **Dependencies**: React, Tailwind CSS v4 `@theme` variables for layout tokens
- **Design Toolkit**: `impeccable-skills` design system parameters

## 🚀 Running Locally

Ensure Node.js and NPM are installed on your environment, then execute:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev

# 3. Compile for production builds
npm run build
```
